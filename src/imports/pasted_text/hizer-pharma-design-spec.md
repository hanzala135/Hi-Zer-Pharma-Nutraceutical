Figma Design Prompt — Hi-Zer Pharma & Nutraceutical Website
Project Overview

Design a premium pharmaceutical and nutraceutical e-commerce website for:

Hi-Zer Pharma & Nutraceutical

The design should contain two completely separate design systems:

Customer Storefront Website
Private Admin Dashboard

Important:

The Admin Dashboard is NOT part of the customer website.
Do NOT add any Admin button, Admin link, hidden menu item, footer link, or navigation item in the storefront design.
The customer website should look like a normal public healthcare e-commerce website.
The Admin Dashboard should be designed as a separate internal application that staff can access manually by typing:
hi-zer.com/admin

into the browser address bar.

There should be no visual connection between the storefront and admin panel.
